# Forçar Login para Conteúdo Restrito

Plugin de controle de acesso que torna o site privado, exigindo login para visualização de conteúdo, desenvolvido pela DSantos Info.

## Descrição

Sistema de "gating" que restringe o acesso ao conteúdo do site, redirecionando usuários não logados para página de login customizada.

## Funcionalidades

- Restrição global para WooCommerce (loja, categorias, produtos, checkout)
- Restrição por página específica via metabox
- Redirecionamento inteligente para login
- Página de login customizada configurável
- Interface administrativa intuitiva

## Dependências

- WooCommerce (para restrições de loja)

## Requisitos

- WordPress 5.0+
- PHP 7.4+

## Instalação

1. Faça upload do plugin para `/wp-content/plugins/`
2. Ative o plugin no painel do WordPress
3. Configure as restrições em Configurações > Login Restrito
4. Defina página de login customizada se necessário

## Configuração

### Restrições Globais
- Ative restrições para loja, categorias, produtos ou checkout

### Restrições por Página
- Use o metabox "Forçar Login" nas páginas individuais
